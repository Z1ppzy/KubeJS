# Конфиги (эталон)

Эти файлы правились вручную в модах и **не деплоятся автоматически** вместе
с папкой kubejs — их нужно копировать отдельно в `config/` инстанса.

Здесь они лежат как эталон, чтобы при скачивании репо на другом устройстве
не потерялись правки баланса. Без них часть работы скриптов не применится.

## Куда копировать

| Файл в репо | Куда в инстансе |
|---|---|
| `_config/solarflux/recipes.cfg` | `config/solarflux/recipes.cfg` |
| `_config/powah.json5` | `config/powah.json5` |
| `_config/mysticalagriculture-common.toml` | `config/mysticalagriculture-common.toml` |

## Что в них изменено

- **recipes.cfg** — отключены 26 дефолтных рецептов Solar Flux (панели 1–8,
  ячейки 1–6, апгрейды, DE-панели). Их заменяют скрипты `15..35_solar_*.js`.
  Зеркало, стёкла и пылающее покрытие оставлены включёнными.
- **powah.json5** — срезана генерация реакторов: blazing 10000→6000,
  niotic 25000→12000, spirited 100000→40000, nitro 500000→150000.
- **mysticalagriculture-common.toml** — `requiresEffectiveFarmland` = true
  (культуры требуют эссенц-землю своего тира). `seedCraftingRecipes` уже был
  false в паке — не трогали.

Перед правкой в инстансе рядом с каждым файлом остались бэкапы `*.bak`.
